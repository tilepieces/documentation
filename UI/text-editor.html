Text editor

The 'Text editor' icon works in the same way as that of the DOM selector, with the difference that, once the element has been selected, it will be possible to edit its textual content.

The writing functionality is based on the contenteditable attribute found on all modern browsers, with the following exceptions:

  • By pressing the "enter" key, a <br> tag will be inserted instead of the default behavior. 
If tilepieces cannot find the selected item in the document source, the item cannot be edited. 

When the button is operated, the DOM selector button will also turn on automatically. When the latter is turned off, the 'text-editor' functionality will also be disabled.
