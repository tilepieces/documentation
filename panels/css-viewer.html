Panels - Css viewer

The panel called "css viewer" allows the modification of the css rules within the stylesheets.
It will only be visible if there is an item selected, otherwise a blank screen will appear.
The rules will be inserted in the "current stylesheet". When inserting a css property, you can select an existing rule or create a new one with a certain selector. See the section "how to insert css rules" for more information.
The panel is divided into 8 sections:
  • Inspector
  • Layout
  • Text
  • Background
  • Border
  • Transform
  • Animations
  • Fonts
