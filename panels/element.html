Panels - Element

This panel is divided into 3 tabs: Main, children and component interface.
At the end of the panel the various parent nodes of the element are always visible. By clicking on one of them you can select the element.

Main

Here you can change the node name, add / remove classes and add / remove attributes.

Children

Displays the child elements of the node. You can select them by clicking on them or move them using the special handle

Component interface

Here, the interfaces of the registered components with the element and its parent nodes are visible.
They can be selected by the appropriate select (if any)
